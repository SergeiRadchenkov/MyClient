Веб-приложение MyClient

Описание: 
Приложение разработано для ведения расписания, данных клиентов и подсчёта заработка и аналитики по клиентам для людей работающих на фрилансе (преподавание, тренировки, встречи). Кроме этого приложение фиксирует оплаченные и неоплаченные встречи, подсчитывает количество отмен по клиентам, а так же может контролировать встречи блоками (если клиент оплачивает встречи блоком, а не разово).

Регистрация и авторизации:
При первом входе в приложение открывается страница авторизации где нужно заполнить Логин - это электронная почта и пароль. Для регистрации нужно будет нажать на ссылку "Регистрация" под кнопкой "Вход". При регистрации нужно будет заполнить данные: Имя, Фамилия, Специальность, Логин (эл. почта) и ввести пароль два раза - все данные для ввода обязательны. После ввода всех данных нужно нажать кнопку "Зарегистрироваться". При авторизации нужно нажать кнопку "Войти". При нажатии этих кнопок происходит переход на главную страницу "Расписание".

Обзор вкладок: в верхней части приложения есть несколько вкладок: "Расписание" (главная страница) - на этой странице указываются все данные о встрече с клиентами, "Клиенты" - здесь заполняются и хранятся все данные о клиентах, "Блоки" - здесь заполняются и хранятся данные о блоках встреч с клиентами, "Профиль" - здесь хранятся данные пользователе, а также формируется аналитика о встречах с клиентами.

"Клиенты" - для начала работы в приложении необходимо завести данные клиентов. При переходе на страницу "Клиенты" в верхней части есть ячейка поиска клиентов. При вводе в ячейку текста выпадает список с соответствующим введёнными буквами. Можно выбрать клиента из выпадающего списка или оставить свой набор текста и нажать значок лупы справа от ячейки. После этого к клиентам применится фильтр и на экране появится клиенты с введёнными данными в ячейке. Правее от ячейки поиска находится ячейка с сортировкой клиентов. По умолчанию стоит сортировка по дате добавления клиента, но можно установить сортировку по имени, по фамилии, по метро, по цене оффлайн, по цене онлайн. Ещё правее от ячейки сортировки находится значок плюса, нажав на который можно будет завести нового клиента. Ниже находится список клиентов, который сортируется по времени добавления клиента: сверху более новые клиенты, ниже более старые. На странице отображается не более 20 клиентов. При добавлении более 20 клиентов внизу экрана добавляется ссылка на следующую страницу с клиентами. Каждый раз при добавлении более 20 клиентов добавляется новая страница. 
При добавлении нового клиента нужно заполнить следующие данные: Имя (единственное обязательное поле ввода), Фамилия, Метро, Улица, Номер дома, Подъезд, Этаж, Квартира, Домофон, Телефон, Цена за работу оффлайн,  Цена за работу оффлайн. После этого нужно нажать кнопку "Сохранить" или "Отмена".

"Блоки" - на данной странице в верхней части есть ячейка поиска клиентов. При вводе в ячейку текста выпадает список с соответствующим введёнными буквами. Нужно выбрать клиента из выпадающего списка. После этого к клиентам применится фильтр и на экране появляются блоки выбранного клиента. Правее от ячейки есть значок крестика, при нажатии на который сбрасывается фильтр по клиенту. Правее от ячейки поиска находится ячейка с выбором статуса блока: "Активные", "Завершённые". 
"Активные" - установлено по умолчанию, при данном статусе на странице показываются только незакрытые блоки. 
"Завершённые" - при данном статусе на странице показываются закрытые блоки. 
Правее от ячейки выбора статуса находится значок плюса, нажав на который можно будет завести новый блок. Ниже находится список блоков, который сортируется по времени добавления блока: сверху более новые блоки, ниже более старые. На странице отображается не более 30 блоков. 
При добавлении нового блока нужно заполнить следующие данные: Клиент (вводятся заведённые данные через поисковую ячейку), Стоимость блока, Количество встреч блока, Количество проведённых встреч блока (на случай если блок заводится после начала проведения встреч в блоке). После этого нужно нажать кнопку "Добавить" или "Отмена".

"Расписание" - главная страница приложения. На данной странице в верхней части есть ячейка поиска клиентов. При вводе в ячейку текста выпадает список с соответствующим введёнными буквами. Нужно выбрать клиента из выпадающего списка. После этого к клиентам применится фильтр и на экране появляются расписание встреч с выбранным клиентом. Правее от ячейки есть значок крестика, при нажатии на который сбрасывается фильтр по клиенту. Правее от ячейки поиска находится ячейка с календарём, по умолчанию установлена текущая дата, но можно вручную установить любую дату, для того, чтобы расписание отображалось начиная с выбранной даты. Правее от ячейки календаря находится значок плюса, нажав на который можно будет завести новую встречу в расписании. Ниже находится расписание на 9 дней. Каждый день это отдельный блок. В нём указывается дата, день недели, карточка с расписанием, где в свою очередь указано: Имя и Фамилия клиента, Время встречи, Стоимость встречи, а также кнопки "Выполнено" - нажимать, когда встреча прошла, "Оплачено" - когда встреча оплачена, "Отменено" - когда встреча отменена. 
Под карточками с расписанием подсчитывается следующая статистика: 
"План выплат" - подсчитывает в карточках за день стоимость встреч, 
"К оплате" - подсчитывает в карточках за день стоимость встреч, где нажата кнопка "Выполнено", но не нажата кнопка "Оплачено", 
"Выплачено" - подсчитывает в карточках за день стоимость встреч, где нажата кнопка "Оплачено", "Отменено" - подсчитывает в карточках за день стоимость встреч, где нажата кнопка "Отменено". 
В каждом блоке встреч за день есть круглая кнопка плюса, при нажатии на которую открывается модальное окно с заведением новой встречу, но с уже проставленной датой.
При добавлении новой встречи нужно заполнить следующие данные: 
Клиент (вводятся заведённые данные через поисковую ячейку), Стоимость встречи, чекбокс "Онлайн" - ставится если встреча будет проходить онлайн, чекбокс "Оплата блоком" - при вводе клиента автоматически проверяется наличие заведённого блока и ставится галочка в данный чекбокс, в блоке при этом будет автоматически закрываться встреча, если галочку убрать, то встреча в блоке закрываться не будет. Далее нужно ввести дату в календаре (если открыли окно через круглую кнопку, дата будет проставляться автоматически) и выбрать время - часы и минуты (по умолчанию стоит 09:00). После этого нужно нажать кнопку "Добавить" или "Отмена".

Все данные Клиентов, Блоков, и Расписания, можно редактировать в карточках. Для этого нужно нажать на карточку, затем на кнопку "Редактировать". Клиентов нужно редактировать в каточку предварительно нажав на шестерёнку в правом верхнем углу карточки, а затем нажать кнопку "Редактировать".

"Профиль" - на данной странице указывается данные пользователя, а именно, Имя и Фамилия, Специальность, Логин (эл. почта). Эти данные можно отредактировать нажав на шестерёнку в правом верхнем углу карточки, а затем нажать кнопку "Редактировать". Под карточкой пользователя находится блок "Незавершенные встречи" - отображается только при наличии незавершенных встреч. В данном блоке указывается: дата и время встречи, имя и фамилия клиента, а также доступны кнопки "Выполнено", "Оплачено", "Отменено". Данные встречи скрываются из блока, если нажать кнопку "Оплачено" или "Отменено" (нажать можно как в этом блоке, так и на странице с расписанием). 
Ниже идёт аналитика по оплатам и встречам с клиентами: 
"Последняя неделя" - сначала идут блоки с общей статистикой за последние 7 дней. В них указана следующая информация: "Завершено встреч" - количество встреч за 7 дней, в которых нажата кнопка "Выполнено", "Отмены" - количество встреч за 7 дней, в которых нажата кнопка "Отменено", "Оплачено" - количество встреч за 7 дней, в которых нажата кнопка "Оплачено", "Оплачено блоков" - количество блоков заведённых за последние 7 дней. 
После блоков общей статистики идёт раздел информации о клиентах за последние 7 дней. В них указывается следующая информация: Имя и Фамилия клиента, Количество встреч, Количество отмен, Оплаты клиента за встречи, Оплаты блоков заведённые за последние 7 дней. Сортировка клиентов происходит по Оплатам клиента за встречи.
"Текущий месяц" - сначала идут блоки с общей статистикой за текущий месяц. В них указана следующая информация: "Завершено встреч" - количество встреч за текущий месяц, в которых нажата кнопка "Выполнено", "Отмены" - количество встреч за текущий месяц, в которых нажата кнопка "Отменено", "Оплачено" - количество встреч за текущий месяц, в которых нажата кнопка "Оплачено", "Оплачено блоков" - количество блоков заведённых за текущий месяц. 
После блоков общей статистики идёт раздел информации о клиентах за текущий месяц. В них указывается следующая информация: Имя и Фамилия клиента, Количество встреч, Количество отмен, Оплаты клиента за встречи, Оплаты блоков заведённые за текущий месяц. Сортировка клиентов происходит по Оплатам клиента за встречи.
"Прошлый месяц" - сначала идут блоки с общей статистикой за прошлый месяц. В них указана следующая информация: "Завершено встреч" - количество встреч за прошлый месяц, в которых нажата кнопка "Выполнено", "Отмены" - количество встреч за прошлый месяц, в которых нажата кнопка "Отменено", "Оплачено" - количество встреч за прошлый месяц, в которых нажата кнопка "Оплачено", "Оплачено блоков" - количество блоков заведённых за прошлый месяц. 
После блоков общей статистики идёт раздел информации о клиентах за прошлый месяц. В них указывается следующая информация: Имя и Фамилия клиента, Количество встреч, Количество отмен, Оплаты клиента за встречи, Оплаты блоков заведённые за прошлый месяц. Сортировка клиентов происходит по Оплатам клиента за встречи.
"Вся статистика" - сначала идут блоки с общей статистикой за всё время накопительно. В них указана следующая информация: "Завершено встреч" - количество встреч за всё время накопительно, в которых нажата кнопка "Выполнено", "Отмены" - количество встреч за всё время накопительно, в которых нажата кнопка "Отменено", "Оплачено" - количество встреч за всё время накопительно, в которых нажата кнопка "Оплачено", "Оплачено блоков" - количество блоков заведённых за всё время накопительно.
